How-To

This application provides a set of functions to access and visualize the proteome and transcriptome data of the manuscript "Scaling of cellular processes with ploidy"
by Yahya et al.

To do so the data is visualized as grouped profile plots which show fold changes of proteome data normalized to the SILAC standard and transcriptome data to a s.pombe spike. For ideal comparability the individual medians of the normalized distributions are shifted to zero (See material and methods for further information).

The visualization is performed with plotly.js, which at any point allows a download of the plot as .png or a transfer to the plotly chart studio (Edit chart >>) to export or further customize the shown data.

Functionality:
Threshold
Enter a log 2 fold change threshold as positive or negative integer or floating point number. The dataset will be filtered for all values that show a total difference between ploidies bigger or equals (positive) or smaller or equals (negative) cutoffs.

Smoothing
Optional: Enter a log 2 fold change smoothing threshold as integer or floating point number in addition to the filtering threshold. All values that show a spike from one to the successive ploidy bigger or equals the (absolute) set smoothing threshold will be excluded from the filtered set.

Biological Process
Enter a Gene Ontology ID of a Biological Process to filter the datasets by annotation. Those can be easily searched for at SGD (linked in navbar) and copy-pasted from there.
Exemplarily, for mitochondrial translation paste GO:0032543, including "GO:", without whitespaces. This function can be combined with a filtering threshold.

Cellular Compartment
Enter a Gene Ontology ID of a Cellular Compartment to filter the datasets by annotation. Those can be easily searched for at SGD (linked in navbar) and copy-pasted from there.
Exemplarily, for cytosolic large ribosomal subunit paste GO:0022625, including "GO:", without whitespaces. This function can be combined with a filtering threshold.

Gene name
Search for a single gene or a comma/semicolon separated list of genes. Gene names are automatically capitalized and completed, eG MCM2-7 can be plotted by searching for "mcm". For a list of all measured gene names see supplement or linked raw data.
